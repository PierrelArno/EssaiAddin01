Aligner une image à gauche et un texte à droite

Consignes (à faire dans Word)

  1. Insère une image et place-la à gauche (ou laisse “Aligné sur le texte”).
  2. Ajoute un paragraphe avec du texte et aligne-le à droite.
  3. Clique Valider : si tout est bon, ❌ passe à ✔️.

🧭 Tutoriel — sans donner la solution

Voici comment procéder dans Word, étape par étape :

  1. Insérer une image
    Onglet Insertion → Images → choisis un fichier. Si elle “flotte”, tu peux la remettre Aligné sur le texte via l’icône d’habillage.
  2. Positionner l’image à gauche
    Laisse l’image à gauche par défaut, ou choisis l’option d’habillage qui colle à la marge gauche.
  3. Ajouter un texte aligné à droite
    Clique dans le document → tape ton texte → onglet Accueil → bouton Aligner à droite.
  4. Vérifier
    Utilise le bouton ✅ Valider ci-dessous. Si besoin, active le bouton ¶ pour voir les marques de mise en forme.

Astuce : si l’image est flottante, elle est quand même reconnue si elle est près de la marge gauche.

❌ Exercice non encore réussi
🔍 Que vérifie la validation ?
  • Au moins une image (inline ou flottante) dans le document.
  • Au moins un paragraphe avec du texte aligné à droite.
  • Si l’image est flottante, elle est détectée si elle est positionnée à gauche.
