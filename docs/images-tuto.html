Tutoriel — Exercice Word : image à gauche et texte à droite

🎯 Objectif

Dans cet exercice, vous devez insérer une image et du texte dans Word, de manière à obtenir une disposition où l’image est placée à gauche et le texte est aligné à droite.

🛠 Préparation

  1. Lancez Word et chargez le complément Images.
  2. Cliquez sur Commencer l’exercice.
  3. Si le document contient déjà du texte, un message vous demandera de le supprimer avant de commencer.

📌 Étapes dans Word

  1. Insérez une image via l’onglet Insertion → Images. Laissez-la en position par défaut (souvent à gauche).
  2. Tapez du texte dans un paragraphe, puis utilisez le bouton Aligner à droite dans l’onglet Accueil.
  3. Cliquez sur Valider dans le complément pour vérifier votre travail.

❌ / ✔️ Validation

Tant que les conditions ne sont pas respectées, l’icône reste ❌. Dès que vous avez correctement inséré l’image et le texte aligné, elle passe en ✔️.

ℹ️ Remarques

  • Utilisez le bouton ¶ pour afficher les marques de mise en forme.
  • L’image peut aussi être flottante, tant qu’elle reste collée à la marge gauche.
